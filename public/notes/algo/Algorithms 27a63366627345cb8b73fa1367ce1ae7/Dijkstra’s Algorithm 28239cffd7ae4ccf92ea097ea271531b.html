Dijkstraâ€™s Algorithm

  1. Use Cases:

It is used to find the shortest distance between all nodes in a graph

NOT USED if graph contains negative weights, because it will fall into an infinite loop since the curr_distance will keep reducing and will always be less than existing distance.

  1. Approach:
  • Initialize a min-heap and a min-distance list (all values in min-distance list are initially inf)
  • Perform a BFS traversal using the min-heap (instead of a queue) to access the closest node first
  • Go to all neighbors of the current node, and if curr_distance < existing distance, set distance[neighbor] to curr_distance. Also add (curr_distance, neighbor) to the min-heap
  • When heap is empty, the min-distance list contains the shortest path from node 1

  1. Complexity

Time complexity: E log V

E: Total number of edges

V: Number of nodes

  1. Code

Assuming we are given a list of edges and their weights, and we have to find the shortest path to all vertices from a particular source node, we can do the following steps:

def shortestPath(self, n: int, edges: List[List[int]], src: int): 
	# create an adjacency list
	adj = defaultdict(list)
	for v1, v2, weight in edges:
		adj[v1].append((v2, weight))
		adj[v2].append((v1, weight)) # assumes the graph is undirected
		
	# function to run dijkstra's algorithm from a particular node
	def dijkstra (node):
		min_heap = [(0, node)]
		distance = [float("inf")] * n # setting the distance to every node as inf
		distance [node] = 0 # distance of node to itself is 0
		
		# performing bfs using heap instead of queue
		while min_heap:
			n = len(min_heap)
			for i in range(n):
				curr_dist, curr_node = heapq.heappop(min_heap)
				for nei, nei_dist in adj[curr_node]:
				
					# if we can reach nei in a shorter distance, 
					# we add it to the heap to explore nodes starting from this node
					if distance[nei] >= curr_dist + nei_dist: 
						distance[nei] = curr_dist + nei_dist
						heapq.heappush(min_heap, (curr_dist + nei_dist, nei))
		return distance
				

# invoke dijkstra on the required source node
return dijkstra(src)
